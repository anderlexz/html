CSS

CSS (Cascading Style Sheets) é uma linguagem usada para definir a aparência e o layout das páginas feita com HMTL, controlando cores, fonte, espaçamentos, tamanhos e posicionamento dos elementos; em resumo enquanto o HTML estrutura o conteúdo, o CSS deixa tudo visualmente organizad e bonito.
