JS

JavaScript é a linguagem responsável por dar comportamen e interatividade ás páginas web, permitindo responder a ações do usuário (como cliquese e digitação), manipular o contúdo do HTML, em tempo real e se comunicar com servidores;

Em resum, enquanto o HTML estrutura e CSS estiliza, o JAvaScript faz a página "funcionar".
